# Project Description

This is a simple Python calculator project. It demonstrates basic arithmetic operations using functions and modular programming.

The project is divided into two files:
- main.py: Runs the program and calls functions
- utils.py: Contains reusable functions like addition and subtraction

## Purpose

The purpose of this project is to practice:
- Code modularity (splitting code into files)
- Basic Git workflow

## How it works

The main file imports functions from utils.py and uses them to perform calculations. The results are then printed to the console.
